### Check Disconnect
This is the deno application that you can see the count of the network disconnection.

## How to use?
1. Install Deno
2. Run `git clone https://github.com/Ogikusa/check-disconnect`
3. Run `deno run --allow-sys main.ts`
4. Enjoy!
